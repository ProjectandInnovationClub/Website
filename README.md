# 🌐 Project & Innovation Club Website

Official website of the **Project & Innovation Club (PIC)**  
📍 *Pimpri Chinchwad College of Engineering & Research (PCCOER)*  


## 🚀 Overview
A modern platform to showcase student innovation, highlight events, and keep the community engaged.

## ✨ Features
- 📅 **Past Events Section** with latest updates  
- 🖼️ **Event Gallery** with images  
- 📢 **Activity Highlights & Announcements**  
- 📱 **Responsive Design (Mobile + Desktop)**  
- ⚡ **Smooth Animations**


## 🛠️ Tech Stack
- ⚛️ React  
- 🎨 Tailwind CSS  
- 🎞️ Framer Motion  

## 🔒 Repository Access
> ⚠️ This repository is **private**

- Access is restricted to authorized contributors

## 📂 Getting Started (Authorized Users Only)

### 1. Clone the Repository
```sh
git clone https://github.com/ProjectandInnovationClub/Website.git
```

2. Navigate to Project Folder
```sh
cd Website
```

3. Install Dependencies
```sh
npm install
```

4. Run Locally
```sh
npm run dev
```
